The Preview Form plugin

Introduction Last updated: 2023-12-25

The preview form for Contact Form 7.

Login

Go to wp-login.php.

Installation

Add New Plugin or Download Plugin

Add New Plugin

1. Go to wp-admin/plugin-install.php → search devui → click Install Now.

2. Click Active.

Download Plugin

1. Go to https://wordpress.org/plugins/ and search devui.

2. Click Preview Form plugin and click Download button.

3. Go to wp-admin/plugin-install.php → click Upload Plugin → click Choose File → click Install Now.

4. Click Active.

Contact Form 7

1. Go to wp-admin/admin.php?page=wpcf7.

2. Click Edit form.
