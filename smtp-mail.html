Introduction Last updated: 2023-12-16

SMTP settings, mail function, send test, save submited data ( phpmailer ). It is very easy to configure and fast.

Login

Go to wp-login.php.

Installation

Add Plugins or Download Plugin

Add Plugins

1. Go to wp-admin/plugin-install.php → search photoboxone → click Install Now.

2. Click Active.

Download Plugin

1. Go to https://wordpress.org/plugins/ and search photoboxone.

2. Click SMTP Mail plugin and click Download button.

3. Go to wp-admin/plugin-install.php → click Upload Plugin → click Choose File → click Install Now.

Settings

Go to wp-admin/options-general.php?page=smtpmail-setting.
