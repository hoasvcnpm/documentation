The Advanced Custom Data plugin

Introduction Last updated: 2023-12-27

The Advanced Custom Data to display anywhere.

Login

Go to wp-login.php.

Installation

Add New Plugin or Download Plugin

Add New Plugin

1. Go to wp-admin/plugin-install.php → search advanced custom data devui → click Install Now.

2. Click Active.

Download Plugin

1. Go to https://wordpress.org/plugins/ and search advanced custom data devui.

2. Click Advanced Custom Data plugin and click Download button.

3. Go to wp-admin/plugin-install.php → click Upload Plugin → click Choose File → click Install Now.

4. Click Active.

Advanced Data

1. Go to wp-admin/edit.php?post_type=acd-data.

2. Click Edit data.
